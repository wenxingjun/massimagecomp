Политика конфиденциальности

Дата вступления в силу: Январь 2026

MassImageComp ("мы", "нас" или "Приложение") — это инструмент для сжатия и обработки изображений для iOS. Мы уважаем вашу конфиденциальность и стремимся защитить вашу личную информацию. Эта Политика конфиденциальности применяется к пользователям по всему миру, за исключением материкового Китая.

Используя это Приложение, вы соглашаетесь с практиками, описанными в этой политике.

1. Информация, которую мы собираем

Мы собираем следующие типы данных для предоставления и улучшения наших услуг:

Идентификаторы

  • ID пользователя: Анонимный UUID, сгенерированный локально для идентификации пользователя и аналитики. Не связан с вашей личностью.
  • ID устройства: Модель устройства, версия операционной системы и идентификаторы устройства для функциональности приложения, аналитики и рекламы третьих сторон (если применимо).

Информация о покупках

  • Платежная информация: Обрабатывается Apple через App Store. Мы не собираем платежные данные напрямую.
  • История покупок: Статус подписки и записи о покупках для функциональности приложения (управление подписками, восстановление покупок).

Данные об использовании

  • Взаимодействие с продуктом: Действия пользователя, такие как просмотры страниц, клики по кнопкам, использование функций и события обработки изображений для аналитики и функциональности приложения.
  • Другие данные об использовании: ID сессии, временные метки, версия приложения, язык, часовой пояс и системная информация для аналитики и функциональности приложения.
  • Рекламные данные: Рекламные идентификаторы и данные о взаимодействии с рекламой для рекламы третьих сторон (если применимо).

Диагностика

  • Данные о сбоях: Логи сбоев, отчеты об ошибках и диагностическая информация для функциональности приложения и аналитики.
  • Данные о производительности: Метрики производительности приложения для функциональности приложения и аналитики.
  • Другие диагностические данные: Дополнительная диагностическая информация для функциональности приложения и аналитики.

Контент пользователя

  • Фотографии или видео: Выбранные изображения для обработки сжатия. Вся обработка происходит локально на вашем устройстве. Мы не загружаем, не храним и не анализируем содержимое фотографий.
  • Электронная почта или SMS: Если вы используете функцию обратной связи, мы можем отправлять электронные письма с отзывами. Адреса электронной почты собираются только если вы их предоставляете.
  • Поддержка клиентов: Отзывы и запросы на поддержку, которые вы отправляете через приложение.

Разрешения

  • Разрешение на библиотеку фотографий: Используется только для выбора изображений и сохранения обработанных результатов. Данные не передаются на наши серверы.
  • Доступ к сети: Используется для рекламы (бесплатная версия), проверки покупок в приложении, аналитики и проверки обновлений.
  • Разрешение на уведомления (необязательно): Используется только для напоминаний о завершении задач.

2. Как мы используем ваши данные

Функциональность приложения

  • Идентификация пользователя и управление подписками
  • Идентификация устройства для безопасности и предотвращения мошенничества
  • Обработка запросов на сжатие изображений
  • Управление подписками и восстановление покупок
  • Предоставление поддержки клиентам
  • Диагностика и исправление сбоев приложения
  • Мониторинг производительности приложения

Аналитика

  • Понимание поведения пользователей и использования функций
  • Оценка эффективности функций продукта
  • Планирование новых функций
  • Измерение размера и характеристик аудитории

Отслеживание данных

Следующие типы данных используются для целей отслеживания:

  • ID пользователя (для аналитического отслеживания)
  • ID устройства (для аналитического отслеживания и отслеживания рекламы третьих сторон)
  • Взаимодействие с продуктом (для аналитического отслеживания)
  • Рекламные данные (для отслеживания рекламы третьих сторон)
  • Другие данные об использовании (для аналитического отслеживания)

Примечание: Отслеживание означает связывание данных пользователя или устройства между приложениями, веб-сайтами или сервисами в рекламных или аналитических целях.

3. Хранение и хранение данных

  • Данные фотографий: Остаются только на вашем устройстве. Мы не храним и не загружаем содержимое фотографий.
  • Статус подписки: Может синхронизироваться через iCloud для функциональности на нескольких устройствах.
  • Аналитические данные: Хранятся на наших серверах (massimagecomp.com) и сохраняются только по мере необходимости для улучшения сервиса.
  • Данные третьих сторон: Рекламные данные обрабатываются службами третьих сторон (например, Google AdMob) в соответствии с их политиками конфиденциальности.
  • Удаление данных: Данные удаляются или анонимизируются, когда они больше не требуются для целей, указанных в этой политике.

4. Службы третьих сторон и обмен данными

Мы делимся данными со следующими третьими сторонами:

Наши собственные серверы

  • Аналитические данные: ID пользователя, ID устройства, взаимодействие с продуктом, данные об использовании, данные о сбоях и данные о производительности отправляются на наши собственные серверы backend (massimagecomp.com) для аналитики и функциональности приложения.
  • Цель: Анализ данных, улучшение приложения и оптимизация сервиса.

Службы Apple

  • App Store: Платежная информация и история покупок обрабатываются Apple для проверки покупок в приложении.
  • iCloud: Статус подписки может синхронизироваться через iCloud для функциональности на нескольких устройствах.

Реклама третьих сторон (только бесплатная версия)

  • Google AdMob: Если вы используете бесплатную версию, мы можем использовать Google AdMob для рекламы. Google может собирать:
    • ID устройства и рекламные идентификаторы
    • Данные о взаимодействии с рекламой
    • Тип устройства и приблизительное местоположение
  • Цель: Отображение рекламы третьих сторон и таргетинг рекламы.
  • Политика конфиденциальности: Политика конфиденциальности Google

Мы не продаем данные пользователей.
