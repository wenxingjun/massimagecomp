【Description】
MassImageComp is a professional image compression and format conversion tool that helps you easily free up storage space on your device while maintaining image quality.

Key Features:
- Smart Batch Compression - Process multiple images at once, significantly reducing file sizes
- Format Conversion - Convert between HEIC, JPEG, PNG, WebP, GIF and other common formats
- Quality Preservation - Intelligent algorithms maintain image clarity during compression
- Local Processing - All processing is done locally on your device for privacy and security
- Batch Processing - Free version supports 3 images, Pro version supports unlimited batch processing
- Multiple Compression Modes - Choose from Smart Compression, High Quality, or Max Compression
- EXIF Data Retention - Option to keep or remove image EXIF information
- Size Adjustment - Set maximum side length with automatic proportional scaling

Pro Version Benefits:
- Unlimited Batch Processing - Process hundreds of images at once
- Ad-Free Experience - Enjoy a clean, distraction-free interface
- All Formats Unlocked - Access to all output format options
- Priority Support - Get priority customer support

All image processing is performed locally on your device. We do not upload, store, or access any of your image data, ensuring complete privacy and security.

The free version can process up to 3 images per batch. Upgrade to Pro to unlock unlimited batch processing and other premium features.
